get_footnotes <- function(vec) {
  list_vec = c()
  for (i in vec){
    myvec <- str_extract_all(i, "\\((.*?)\\)")
    list_vec <- append(list_vec, myvec)
  }
  return(list_vec)
}

get_nums <- function(vec){
  vec <- str_replace_all(vec, "[()]", "")
  vec <- str_replace_all(vec, "\\,", "")
  vec <- str_replace_all(vec, "[:alpha:]", "")
  vec <- str_replace_all(vec, "\\-", "")
  vec <- str_trim(vec, side ="right")
  print(vec)
  df<- data.frame(
    First.Num = str_match(vec, "[^[:blank:]]+"),
    Last.Num = str_sub(vec, -4, -1 )
  )
  return(df)
}

When talking about minimum wage by region, it is important to talk about the difference between highest and lowest values of the minimum wage. This graph provides a strong visual of the difference in minimum wage by region throughout the past 53 years In general, a smaller difference is desired, meaning less of a disparity in minimum wage. We can see that the gap is growing in the Midwest, South, and the West. But the Northeast is experiencing a low disparity between states in the region. This can be interpreted in many ways. We know from the graph above that the Northeast is drastically increasing their minimum wage amounts. So by comparing the two graphs, we can see that the northeast is doing something right by lowering the disparity and raising the amount.
